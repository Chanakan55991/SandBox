## About
SandBox is a simple programming language made with JS & ANTLR4.

Still in development, stay tuned!

## Examples

- A simple example of if else statement.
```
let a = 3;

if a > 5 {
    print "Hi";
} else if a < 5 {
    print "Hello"
} else {
    print a;
}
```

- A simple example of while statement.
```
let a = 1;

while a < 4 {
    let b = 0;
    while b < 3 {
        print a + b * 3;
        let b = b + 1;
    }
    let a = a + 1;
}
```
