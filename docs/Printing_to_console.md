# Printing to console

## Printing string
```
print "Hello World!";   // Hello World!
```

## Printing integer
```
print 123;    // 123
```

## Printing double
```
print 1.23;   // 1.23
```

## Printing boolean
```
print true;   //true
```

## Printing variable
```
let a = "Hello";
let protected b = 123;

print a;    // Hello
print b;    // 123
```
