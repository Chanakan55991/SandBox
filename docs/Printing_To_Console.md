# Printing To Console

## Printing String
```
print("Hello World!");   // Hello World!
```

## Printing Integer
```
print(123);    // 123
```

## Printing Float
```
print(1.23);   // 1.23
```

## Printing Boolean
```
print(true);   //true
```

## Printing Variable
```
let a = "Hello";
let protected b = 123;

print(a);    // Hello
print(b);    // 123
```

## Printing Expression
```
print("Hello " + "World");   // Hello World
print(1 + 2);                // 3
print(123 == 123);           // true
```
