# Declaring variables
## Unprotected variables
```
let a = "Hi";   // String
let b = 123;    // Integer
let c = 1.23    // Float
let d = true    // Boolean

let e = 1 + 2;                // Expression
let f = "Hello " + "World";   // Expression
let g = 123 != 12345;         // Expression
```

## Protected variables
```
let protected a = "Hi";
let a = 123;    // Error
```
