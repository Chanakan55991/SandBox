# If else statement

## Usage
```
if <statement> {
    ...
} else if <statement> {
    ...
} else {
    ...
}
```
``else if`` and ``else`` is optional.
You can put as many ``else if`` as you want, and one or no ``else``.

## Example
```
let a = 3;

if a > 3 {
    print "a is greater than 3.";
} else if a < 3 {
    print "a is smaller than 3.";
} else {
    print "a is equal to 3.";
}
```
